# CHBIM 功能设计

## 核心概念

### 1. 预设与族

基于类型学的预设体系（类似 Revit 族）。

预设 = 某一类型建筑构件的模板，包含几何规则和参数定义。

### 2. 参数属性

混合 Revit 属性面板 + SolidWorks 设计树的参数管理方式。

### 3. 数据统计与查询

基于簇（组合）和实体（构件）两级的数据统计。

### 4. 几何关系

参考 SolidWorks 装配体的约束体系。

- **主动与随动**：几何层面和数值层面的关联
- **公式驱动**：随动数值通过公式确定

### 5. 查看与编辑

选择过滤器支持多粒度选择：点、线、面、实体、簇。
