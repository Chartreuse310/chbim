| 阶段    | 内容                    | 状态                 |
| ----- | --------------------- | ------------------ |
| 编码体系  | 基于"缝"的构件编码——每个构件叫什么   | 进行中：v1.1 以镇国寺万佛殿为例 |

路径：`src/`、项目根目录

| 阶段        | 内容                                | 状态  |
| --------- | --------------------------------- | --- |
| 基础        | C++ 项目构建                          | 待开始 |

## 里程碑

1. **M1 — 编码体系**— 至少一个完整案例的"缝"编码
2. **M2 — 参数化原型** — 营造法式核心参数可驱动几何生成
3. **M3 — MVP** — 参数输入 → OCCT 几何 → 3D 可视化
4. **M4 — AI 辅助** — Python Agent 可调用几何内核，自然语言驱动建模
