# 基于"缝"的构件编码体系（v1.1）

> 旧版：[v1.0](v1.0.md)

目标：

- 构件编码的唯一性
- 与文字的映射（保证能转化为唯一性文字）

| 类型     | 描述                        |
| ------ | ------------------------- |
| 朝向     | F：正；B：背；L：左；R：右           |
| 内外     | i：内；o：外                   |
| 位置     | 0+：距离中线的位置                |
| 缝：柱头平面 | CAO：槽缝；PUZUO：铺作缝；TIAO：跳心缝 |
| 缝：梁架   | FU：栿缝；TUAN：槫缝             |
| 构件     | 名称拼音小写+横纵定义，如tuan         |

- 不同类型间用 `-` 相连，同类型直连，同方向不能出现重复定义。如 `LoL1` 表示左侧最外至左一，`Lo-o1` 表示左侧最外的外一。
